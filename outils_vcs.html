Les VCS sont parfois connus sous le nom d'outils de gestion du code source (SCM) ou de systèmes de contrôle des révisions (RCS). Git est l'un des outils VCS les plus populaires utilisés aujourd'hui. Git est un VCS distribué, une catégorie connue sous le nom de DVCS, dont nous parlerons plus loin.
